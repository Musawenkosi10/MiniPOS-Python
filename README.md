## MiniPOS Python Project 🛒

A **console-based Point-of-Sale (POS) system** built with Python OOP, MySQL, CSV, Pandas, and Matplotlib.  
This project simulates a real grocery store checkout and tracks sales data with analytics.

---

## Features

- Cashier & Customer input  
- Add multiple products per transaction  
- Automatically calculates totals & change  
- Saves sales to **CSV** and **MySQL database**  
- Run **sales analytics** with charts  
- Beginner-friendly but **portfolio-ready**
